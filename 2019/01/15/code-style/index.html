码风整改计划
2019-01-15|闲聊计划

背景

最近在认真研读蓝书（即《算法竞赛入门经典—训练指南》），对汝佳佩服的五体投地的同时，我也被他的码风所倾倒。也因此萌生了改码风的念头。

我之前的码风……实在是不敢恭维，主要是太过紧凑，阅读比较困难。说白了就是我不爱加空格……

整改目标

有问题就要改
在这里贴一下整改目标。

大括号

大括号不换行，左大括号前加空格
e.g.
for(int i = 1; i <= n; ++i) {
int main {

运算符

  • 赋值运算符、比较运算符、逻辑运算符，除’$!$’外，两侧一般加空格
    e.g.
    int a = 1;
    if(a == 1 || !a)

  • 当普通运算符只被大括号包含的时候，两侧才加空格
    e.g.
    if(1){ a += 1; }
    a[1+1]

  • 左移右移、自增自减一般不加空格(左移右移+cin/cout除外)
    e.g.
    1<<31; ++a;
    cin >> a;

其他

  • 头文件加空格
    e.g.
    #include <algorithm>

  • 若分号和逗号后有语句，必加空格。
    e.g.
    int a = 1, b = 2
    for(int i = 1; i <= n; ++i)

  • STL容器和类型之间加空格
    e.g.
    vector <int> a;

  • 尽量避免压行

  • 在某些特殊情况下，可以无视以上规则(防止真香)
